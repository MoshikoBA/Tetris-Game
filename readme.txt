Highlights for the game :

The joker does not trample existing shapes, but skips them if there is empty space .
The joker stops moving only in two cases:
1. There is no empty space below.
2. The user clicks 's'.

The score is according to the instructions.

Move the shapes only by the arrow keys.
The beginnig sleep speed is 200 , and limited between 50-500.

Each shape has different color.
There is no cursor on the screen.

In this exercise the Tetris class has a pointer to Shape class as opposed to EX1.
